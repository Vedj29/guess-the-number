
#  Guess The Number (Python)

A simple console-based number guessing game built using Python.  
The player tries to guess a secret number, and the program provides hints until the correct guess is made.

---

##  Features

- User input handling
- Infinite loop using `while True`
- Conditional statements (`if-elif-else`)
- Feedback messages:
  - Too low
  - Too high
  - Correct guess
- Loop termination using `break`

---

## How the Game Works

1. A secret number is predefined in the program.
2. The user is asked to guess the number.
3. The program compares the guess with the secret number.
4. The loop continues until the correct number is guessed.

---

##  Technologies Used

- Python 3
- Basic programming concepts:
  - Loops
  - Conditionals
  - User input
  - Type conversion
